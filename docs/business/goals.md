# Cele Biznesowe - FocusFlow

## Misja
Umożliwić osobom z ADHD i ambitnym planistom maksymalne ułatwienie priorytetyzowania codziennych wyzwań.

## Główne Cele

### 1. Inteligentna Macierz Eisenhowera
- Dynamiczny podział zadań na 4 ćwiartki
- Hard-limit 5 zadań w Q1 (pilne + ważne)
- Brain Dump quiz (6 pytań) dla klasyfikacji

### 2. 7-Trybowy Timer Focus
- Pomodoro `25/5`, Deep Work `50/10`/`90/15`, Flow `5/0`
- State machine niezależny od React
- Wake Lock API dla blokowania wygaszania

### 3. Auto-Eskalacja Q2 → Q1
- Codzienne sprawdzanie terminów
- Automatyczne przenoszenie ważnych zadań

### 4. Offline-First PWA
- IndexedDB jako baza danych lokalna
- BroadcastChannel dla synchronizacji zakładek
- Instalacja jak aplikacji natywnej
