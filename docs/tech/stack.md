# Stack Technologiczny FocusFlow 2.0

> Single Source of Truth dla technologii użytych w projekcie

## Core Stack

| Warstwa | Technologia | Wersja | Uzasadnienie |
|---------|-------------|--------|--------------|
| Framework | React | 18.x | Komponentowy model, ecosystem |
| Język | TypeScript | 5.x | Typowanie, bezpieczeństwo |
| Styling | Tailwind CSS | 3.x | Utility-first, mobile-first |
| Build Tool | Vite | 5.x | Szybki HMR, modern output |
| Storage | Dexie.js | 3.x | Wrapper na IndexedDB, offline-first |

## Dodatkowe Zależności

| Pakiet | Cel |
|--------|-----|
| lucide-react | Ikony |
| clsx + tailwind-merge | Dynamiczne klasy CSS |

## Wymagania Środowiskowe

- Node.js >= 18
- npm >= 9

## Konfiguracja Tailwind

- Custom theme z neonową paletą
- Mobile-first breakpoints
- Ekstensywna konfiguracja colors w `tailwind.config.js`
